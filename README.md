# 📌 ClientControl System

Sistema web de gerenciamento de clientes com operações completas de **CRUD (Create, Read, Update, Delete)**, desenvolvido utilizando PHP e MySQL.

---

## 🚀 Sobre o projeto

O **ClientControl System** é uma aplicação web que permite gerenciar clientes de forma prática e eficiente.

O projeto foi criado com foco em aprendizado e prática de desenvolvimento web, abordando conceitos fundamentais como:

- Integração com banco de dados
- Manipulação de formulários
- Organização de sistemas web
- Operações CRUD completas

---

## 🛠️ Tecnologias utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP (ambiente local)


---

## ⚙️ Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Edição de dados
- Exclusão de registros
- Sistema de login (administrador)
- Dashboard com informações gerais
